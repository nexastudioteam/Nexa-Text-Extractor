# Nexa-Text-Extractor
Nexa Text Extractor is a fast web-based OCR tool that extracts text from images and instantly translates it to Arabic with one click.

# Nexa Text Extractor

---

## العربية

Nexa Text Extractor هو تطبيق ويب يعتمد على تقنية OCR لاستخراج النصوص من الصور باستخدام Tesseract.js، مع دعم ترجمة فورية للنصوص المستخرجة إلى اللغة العربية.

التطبيق لا يعتمد على نظام رفع ملفات تقليدي، بل يدعم:
- سحب وإفلات الصور داخل الصفحة
- لصق الصور مباشرة من الحافظة (Ctrl + V)

بعد معالجة الصورة يتم استخراج النص تلقائيًا، مع إمكانية ترجمته أو نسخه بسهولة.

---

## المميزات

- سحب وإفلات الصور (Drag & Drop)

- دعم الوضع الفاتح والداكن

## English

Nexa Text Extractor is a web-based OCR application that extracts text from images using Tesseract.js, with real-time translation support into Arabic.

The application does not use traditional file upload methods. Instead, it supports:
- Drag & Drop image input
- Direct paste from clipboard (Ctrl + V)

After processing the image, text is automatically extracted with options to translate or copy it easily.

---

## Features

- Drag & Drop image support
- Paste images directly from clipboard
- OCR text extraction using Tesseract.js
- One-click translation to Arabic
- One-click copy functionality
- Automatic detection of codes and numbers
- Clean and responsive interface
- Light & Dark mode support

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Tesseract.js
- Google Translate API

---

## Usage

1. Drag & drop an image or paste it directly (Ctrl + V)
2. Wait for OCR processing
3. Use translate or copy features as needed
